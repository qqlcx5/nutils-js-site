# max数组中最大值

过滤原数组中所有的非假值元素，返回数组中的最大值

max(array)

参数

  • array待处理的数组

示例

max([0, -1, -2, -3, false])
// => 0

源码

const max = arr => Math.max(...arr.filter(v => Boolean(v) || v === 0))
