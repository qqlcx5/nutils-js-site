# `have`数组集合

创建一个共有的`array`值的数组，每个值包含在其他给定的数组中。

**参数**

-   `array` 要检查的数组
-   `array2`要包含的数组

**例子**

```js
have([1, 2, 6, 7], [1, 2, 9, 5])
// => [ 1, 2 ]
```

**源码**

```js
const have = (a, b) => {
    const s = new Set(b)
    return a.filter(x => s.has(x))
}
// ES6 includes
const have = (arr, values) => arr.filter(v => values.includes(v))
```
