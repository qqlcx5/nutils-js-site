🚀 nutils-js 是一套在开发中实践积累起来的实用 JavaScript 工具库。

作为Web开发人员，在日常开发中难免遇到Array Object String Number等数据处理，或者是防抖节流函数等性能优化亦或者URL参数处理 类型判断等等这些都是使用率较高的 JavaScript 模块，避免在不同项目中复制粘贴的麻烦，发布在 npm 上，如果后续有更优的写法也会做进一步的更新。还有就是你有好的建议或者想为本项目贡献一份力的话，欢迎为本项目提交 pr，一起探讨和交流。

# ⬇️ 安装

$ npm i --save nutils-js

# 🍻 使用

const nutils = require('nutils-js')
nutils.chunk([1, 2, 3], 2)
