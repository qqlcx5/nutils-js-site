# Javascript 前端高效开发工具库

(opens new window) (opens new window)

🚀 nutils-js 是一个模块化、高性能的 JavaScript 实用工具库。

日常开发中难免遇到Array Object String Number等数据处理，或者是视图上防抖节流函数等性能优化亦或者URL参数处理``类型判断等等为避免不同项目复制粘贴的麻烦，将方法进行封装，发布在 npm 上，如果后续有更优的写法也会做进一步的更新。还有就是你有好的建议或者想为本项目贡献一份力的话，欢迎为本项目提交 pr，一起探讨和交流。

# ⬇️ 安装

$ npm i --save nutils-js

# 🍻 使用

const nutils = require('nutils-js')
nutils.multArray([1, 2, 3], 2)
